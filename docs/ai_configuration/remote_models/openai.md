# OpenAI API Key Setup

Use this guide to create an OpenAI API key and connect it in Sulla Desktop.

## Create your OpenAI key

1. Open the OpenAI API keys page: https://platform.openai.com/api-keys
2. Sign in to your OpenAI account.
3. Create a new API key.
4. Copy the key and store it securely.

## Add key in Sulla

1. Open main Electron menu.
2. Open **Language Model Settings**.
3. Go to **Models > Remote Models**.
4. Select **OpenAI**.
5. Paste your API key and save.

After saving, available OpenAI models should appear in the model picker.
