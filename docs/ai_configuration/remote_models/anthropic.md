# Anthropic API Key Setup

Use this guide to create an Anthropic API key and connect it in Sulla Desktop.

## Create your Anthropic key

1. Open Anthropic Console: https://console.anthropic.com/
2. Sign in to your Anthropic account.
3. Create a new API key.
4. Copy the key and store it securely.

## Add key in Sulla

1. Open main Electron menu.
2. Open **Language Model Settings**.
3. Go to **Models > Remote Models**.
4. Select **Anthropic**.
5. Paste your API key and save.

After saving, available Anthropic models should appear in the model picker.
