# Grok (xAI) API Key Setup

Use this guide to generate an xAI API key for Grok and connect it in Sulla Desktop.

## Create your xAI key

1. Open the xAI API Keys page: https://console.x.ai/team/default/api-keys
2. Sign in (or create an xAI account).
3. Create a new API key.
4. Copy the key and store it securely.

## Add key in Sulla

1. Open main Electron menu.
2. Open **Language Model Settings**.
3. Go to **Models > Remote Models**.
4. Select **Grok (xAI)**.
5. Paste your API key and save.

After saving, Grok models should appear in the model picker.
