# Google Gemini API Key Setup

Use this guide to create a Gemini API key and connect it in Sulla Desktop.

## Create your Gemini key

1. Open Google AI Studio API keys: https://aistudio.google.com/app/apikey
2. Sign in with your Google account.
3. Create a new API key.
4. Copy the key and store it securely.

## Add key in Sulla

1. Open main Electron menu.
2. Open **Language Model Settings**.
3. Go to **Models > Remote Models**.
4. Select **Google Gemini**.
5. Paste your API key and save.

After saving, available Gemini models should appear in the model picker.
