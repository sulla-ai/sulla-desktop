# System Requirements Check

Use this checklist before installing Sulla Desktop.

Source: [Sulla Desktop Downloads](https://sulladesktop.com/downloads)

## Operating System Requirements

- **Windows**: Windows 10 or later (64-bit)
- **macOS**: macOS 11 Big Sur or later
- **Linux**: Ubuntu 20.04+, Fedora 34+, or Debian 11+

## Hardware Requirements

### Minimum

- **Processor**: Dual-core 2.0 GHz or equivalent
- **Memory**: 4 GB RAM
- **Storage**: 2 GB available disk space
- **Network**: Broadband internet connection

### Recommended

- **Processor**: Quad-core 3.0 GHz or Apple M1/M2
- **Memory**: 8 GB RAM or more
- **Storage**: 5 GB available SSD space
- **Network**: High-speed internet (recommended for AI features)

## Pre-Install Checklist

- [ ] My OS version is supported
- [ ] I have at least the minimum CPU/RAM/storage
- [ ] I have a stable internet connection for setup and cloud AI features
- [ ] I selected the correct installer for my platform (`.exe`, `.dmg`, `.AppImage`, `.deb`, or `.rpm`)

## Platform Notes

- **Windows**: Native performance, auto-updates, Windows Hello support
- **macOS**: Universal binary (Intel + Apple Silicon)
- **Linux**: AppImage supported across major distros, plus `.deb` and `.rpm` options
